# Twitter Sentiment Analysis using NLP

## Project Overview
This project performs sentiment analysis on Twitter data to classify tweets as positive, negative, or neutral using Natural Language Processing techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn

## Key Features
- Text preprocessing (cleaning, tokenization, stopword removal)
- Feature extraction using CountVectorizer / TF-IDF
- Machine Learning model training
- Sentiment classification
- Data visualization

## Results
Achieved strong accuracy in predicting tweet sentiment.
